Planner is a Calendar/To-Do app.

Features:
  - Personal View
    + Schedule Pane
    + Task Backlog Pane
      - Display existing unassigned tasks
      - Allow assigning/scheduling tasks
      - Drag-n-drop From Backlog to Schedule
  - Family View
    + Schedule Pane Per Person
      - Drag-n-drop between people to reassign/reschedule
  - Gamified
    + Tasks can have point values
